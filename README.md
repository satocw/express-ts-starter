# 使い方

VSCode でデバッグ

 - Launch Program を実行

VSCode なしで実行

 - npm start

VSCode なしでデバッグ

 - npm run debug
 - chromeのDevTools for Node.jsを開く

# URL

GET /
  FLEXSCHE GP の最新バージョン

GET /comm
  FLEXSCHE Communicator の最新バージョン

GET /gp_old_releases
  FLEXSCHE GP の過去（メンテナンス非対応）バージョン

GET /comm_old_releases
  FLEXSCHE Communicator の過去（メンテナンス非対応）バージョン
